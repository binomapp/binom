CLI usage example:

BNQL

BinOM Node Query Language

Example:

[member_select_command]/+[add_command]/-[remove_command]/=[set_command]/?[selector_command]/<[get_command];

name/0/+obj{a:arr[arr8[1,2,3],arr16[1,2,3],arr32[1,2,3],arr64[1,2,3]],b:8[1],c:16[2],d:32[3],e:64[4]}/-0/-name/=arr8[1,2,3]/?name.0=8|name.1=$obj{8(127),(255)}/<0/<name/<

Command symbols:

  • without symbol - member select command. Varieties:
    • name - select member with name name
    • 0 - select member with index 0
  • + - add member command. Varieties:
    • +obj{id:64(0),login:arr8[admin],password:arr8[admin],access_lvl:8(0)} - add to array structure: 
      [array]{
| ...
|[object]:{
||"id":[qword]:0 
||"login":[byte array]:97 100 109 105 110  
||"password":[byte array]:97 100 109 105 110  
||"access_lvl":[byte]:0 
|}
}
    • +file_name:arr8[Hello_world] - add to object structure: 
      [object]{
| ...
| "file_name":[byte array]:72 101 108 108 111 95 119 111 114 108 100
}
    • +1,2,3 - add elements 1,2,3 to primitive array
  • - - remove command. Varieties:
    • -name - remove member with name name
    • -0 - remove member with index 0
  • = - set command. Varieties:
    • =obj{id:64(1),login:arr8[guest],password:arr8[guest],access_lvl:8(255)} - set to variable structure: 
      [object]:{
|"id":[qword]:1 
|"login":[byte array]:103 117 101 115 116  
|"password":[byte array]:103 117 101 115 116  
|"access_lvl":[byte]:-1 
}
    • =127 - set value to primitive variable
    • =1,2,3 - set values to primitive array (Warning: type casting is used based on the size of numbers!)
  • < - get command. Varieties:
    • < - get current selected variable
    • <0 - get member with index 0 of current selected variable
    • <name - get member with name name of current selected variable

Definitions:

  • BinOM (Binary Object Model) - format of serialization, deserialization of data
  • BinOMDB (Binary Object Model DataBase) - hierarchical database management system in BinOM format
  • BinOMQL (Binary Object Model Query Language) - declarative programming language for modifying and manipulating BinOM data
